UC-0002 Reportar incidencia
Versión 1.0 ( 05/12/2016 )
Autores
  • Alejandro Gómez
Fuentes ?
Dependencias Ninguno
Descripción El sistema deberá comportarse tal como se describe en el siguiente caso de uso cuando el cliente desee reportar una incidencia sobre su instalación
Precondición PD
Secuencia normal Paso Acción
1 El actor Cliente (ACT-0001) indica al sistema en qué instalación se ha producido la incidencia
2 El actor Cliente (ACT-0001) indica la estancia en la que se ha producido la incidencia:
- Habitación
- Oficina
- Sala especializada
- Otros
3 El actor Cliente (ACT-0001) introduce una descripción de la incidencia
4 El actor Cliente (ACT-0001) indica la hora a la que se produjo la incidencia
5 El sistema registra la hora a la que se reportó la incidencia
6 El sistema le asigna un código identificativo único a la incidencia
7 El sistema marca el estado de la incidencia como "Reportada"
Postcondición PD
Excepciones Paso Acción
3 Si es necesario, el actor Cliente (ACT-0001) podrá cambiar la descripción de la incidencia en cualquier momento, a continuación este caso de uso continúa

UC-0003 Tramitar incidencia
Versión 1.0 ( 05/12/2016 )
Autores
  • Alejandro Gómez
Fuentes ?
Dependencias Ninguno
Descripción El sistema deberá comportarse tal como se describe en el siguiente caso de uso cuando el reporte de una incidencia le llega al gestor
Precondición PD
Secuencia normal Paso Acción
1 El actor Gestor de incidencias (ACT-0002) suministra el código de la incidencia
2 El sistema muestra los reparadores dados de alta
3 El actor Gestor de incidencias (ACT-0002) selecciona un reparador dado de alta para trabajar esta incidencia
4 El sistema marca el estado de la incidencia como "Asignada"
5 El sistema comunica al reparador que tiene una incidencia por resolver
Postcondición PD
Excepciones Paso Acción
1 Si el gestor no posee el código de la incidencia, se realiza el caso de uso Buscar incidencia (UC-0004) , a continuación este caso de uso continúa
1 Si es necesario ampliar la descripción que proporcionó el cliente, se realiza el caso de uso Modificar incidencia (UC-0005) , a continuación este caso de uso continúa
4 Si es necesario cambiar el estado de la incidencia, se realiza el caso de uso Modificar incidencia (UC-0005) , a continuación este caso de uso continúa
5 Si es necesario cambiar el reparador asignado, se realiza el caso de uso Modificar incidencia (UC-0005) , a continuación este caso de uso continúa
Rendimiento Paso Tiempo máximo
