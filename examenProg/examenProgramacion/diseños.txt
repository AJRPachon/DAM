El primer diseño que voy a plantear es el que voy a implementar:

Cuatro clases y el main, una UnidadSalida, una Pantalla, Fecha y la gestora.

La UnidadSalida tendrá las siguientes propiedades:

- identificador		tipo int
- nombreFabricante	tipo String
- numeroSerie		tipo int
- unidades			tipo int
- precio			tipo double

La clase Pantalla tendrá:

- Todos los atributos de unidadSalida
- resolucion		tipo int
- tipoTecnologia 	tipo enum
- fecha				tipo Fecha (implementada por mi)

Pantalla heredará de unidadSalida puesto que comparten todos los atributos de esta.

Habrá una gestora para tratar con los archivos binarios e implementar las funcionalidades 
de este programa.
